# お薬飲んだボタン (Good Dose Drug Button)

お薬を飲んだ自分を全力で褒めるためのWebアプリケーションです。

## 🔗 サービスリンク

[**💊お薬飲んだボタン - GitHub Pages**](https://neko3cs.github.io/good-dose-drug-button/)

## 🌟 特徴

- **褒めてくれる**: ボタンを押すと、ランダムな20種類の褒め言葉であなたを肯定します。
- **X (Twitter) 連携**:「お薬を飲みました。えらい！」というメッセージを簡単にXで共有できます。
- **目に優しいデザイン**: 落ち着いた緑色のテーマと、押しやすい巨大なボタンを採用しています。

## 🛠 テックスタック

- **Framework**: Angular v21
- **UI Components**: Angular Material
- **Styling**: Tailwind CSS
- **Deployment**: GitHub Pages (docsフォルダー)
